## 你的首页需要包含的模块

**占据整个视口高度的首屏展示**

## 设计风格建议

### 配色方案
```css
/* 主色调 - 科技感蓝紫渐变 */
--primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 深色背景 */
--bg-dark: #0f172a;
--bg-card: #1e293b;

/* 文字 */
--text-primary: #f8fafc;
--text-secondary: #94a3b8;

/* 强调色 */
--accent: #22d3ee;
--success: #22c55e;
```

### 动效建议
- **Hero 区域**：渐变背景动画 / 粒子效果 / 3D 模型展示
- **卡片悬停**：轻微上浮 + 阴影加深
- **滚动动画**：元素进入视口时淡入上移
- **按钮**：悬停时 scale(1.05) + 发光效果

### 响应式断点
- **移动端**：<768px，单列布局，导航改为汉堡菜单
- **平板**：768px-1024px，两列套餐卡片
- **桌面**：>1024px，完整三列布局

## 技术实现建议

由于是营销首页，可以考虑：

1. **SSR/SSG**：使用 Next.js 实现服务端渲染，利于 SEO
2. **或静态页面**：首页可以是纯静态 HTML，用户后台用 React SPA
3. **CDN 加速**：首页静态资源部署到 CDN
4. **性能优化**：图片懒加载、代码分割、压缩

# 订阅管理 API

处理用户订阅的全生命周期管理，包括查询、升级、取消、续费等操作。

## 基础路径

`POST /api/v1/subscriptions`

## 认证

所有接口均需要 Bearer Token 认证。

## 主要功能

- **获取订阅**: 查询当前有效订阅和历史订阅记录
- **升级订阅**: 支持立即升级（补差价）或下个周期升级
- **取消订阅**: 支持到期取消或立即取消
- **续费管理**: 自动续费开关控制和手动续费

## 错误码

| 错误码 | 说明 |
|--------|------|
| `NO_ACTIVE_SUBSCRIPTION` | 用户当前没有有效订阅 |
| `PLAN_NOT_FOUND` | 指定的套餐不存在 |
| `INVALID_UPGRADE` | 升级操作无效（如降级或同级切换） |
| `INTERNAL_ERROR` | 服务器内部错误 |
